1 项目与工程管理基础

什么是工程项目？
以建筑物、构筑物、基础设施等为交付成果，具有一次性、目标性、系统性、周期性的建设任务。
工程项目三大核心目标是什么？
质量目标、进度目标、成本目标（俗称"三驾马车"）。
工程项目最本质的特征是什么？
一次性、唯一性，不可重复。
项目全生命周期分为哪几个阶段？
决策阶段 → 设计阶段 → 实施施工阶段 → 竣工验收阶段 → 运营维护阶段。
工程管理的核心内涵是什么？
对项目全生命周期的质量、进度、成本、安全、风险、资源进行全方位统筹管控，保障项目目标落地。
项目管理者的使命是什么？
通过计划、组织、指挥、协调、控制，对有限资源（时间、成本、质量）进行优化配置和充分利用，满足利益相关者需求。
项目管理五大要素是什么？
客体（项目）、主体（管理者）、目的（达成目标）、职能（计划/组织/指挥/协调/控制）、依据（有限资源）。

2 进度管理 计算题重点

常用进度管理工具有哪些？
横道图（甘特图）、双代号网络图、单代号网络图。
进度管理的主要目的？
合理安排工序，保障项目按期完工。
关键路径的定义？
网络图中总持续时间最长的线路，决定项目总工期。
关键工作的特点？
位于关键路径上，总时差为 0，一旦延误直接导致总工期延长。
总时差的定义？
在不影响项目总工期的前提下，工作可拖延的机动时间。

3 质量管理

工程质量控制三阶段？
事前控制 → 事中控制 → 事后控制。
工程质量影响因素有哪些？
人、材、机、法、环（人员、材料、机械、方法、环境）。

4 成本管理

成本管理的核心原则？
全过程管控、全员管控、动态管控。
成本管理包含哪些内容？
成本预测 → 成本计划 → 成本控制 → 成本核算 → 成本分析与考核。

5 安全管理

安全管理方针是什么？
安全第一、预防为主、综合治理。

6 风险管理 简答题高频

风险的特点？
不确定性、客观性、可预测性、可管控性。
常见工程项目风险有哪些？
进度风险、成本风险、质量风险、安全风险、政策风险、自然风险、市场风险等。
四种风险应对方式？
风险规避 · 风险减轻 · 风险转移 · 风险自留。
风险管理步骤？
  1. 风险识别：找出所有潜在风险。
  2. 风险分析与评估：判断概率和损失程度，分级。
  3. 风险应对：制定规避、减轻、转移、自留等措施。
  4. 风险监控：全过程动态跟踪，及时调整。

7 项目组织

常见组织结构形式？
直线制、职能制、矩阵制。
矩阵制组织结构的优缺点？
✅ 优点：资源利用率高、组织灵活性强、各专业沟通协作高效。
⚠️ 缺点：存在多头领导，员工易接收多重指令，管理协调难度较大。

8 招投标与合同

招投标的基本原则？
公开、公平、公正、诚实守信。
常见工程合同类型？
总价合同 · 单价合同 · 成本加酬金合同。

🧮 第二部分：计算题专攻 —— 网络图与关键路径

核心知识点

关键路径
网络图中总持续时间最长的线路。
关键工作
关键路径上的工作，总时差 = 0，延误即延长总工期。
总工期
关键路径的总持续时间。

核心公式

EF = ES + D

LS = LF - D

TF = LS - ES = LF - EF

EF：最早完成  |  ES：最早开始  |  D：持续时间
LS：最迟开始  |  LF：最迟完成  |  TF：总时差

通用解题步骤

  1. 梳理所有线路：找出从起点到终点的所有可能路径。
  2. 计算每条线路工期：将路径上所有工作的持续时间相加。
  3. 确定关键路径与总工期：工期最长的那条线路就是关键路径，其持续时间即为项目总工期。
  4. 计算工作时差（如有要求）：正推法算最早时间 (ES, EF)，逆推法算最迟时间 (LS, LF)，最后用公式算总时差。

📝 例题演练

假设某项目各工作持续时间：A=3天, B=2天, C=4天, D=3天, E=5天

网络图逻辑关系：

线路1：A → C → E

线路2：A → B → D → E

解题过程：

  • 线路1工期：3 + 4 + 5 = 12天
  • 线路2工期：3 + 2 + 3 + 5 = 13天

∵ 线路2的工期(13天) > 线路1的工期(12天)
∴ 关键路径：A → B → D → E
∴ 项目总工期：13天

关键工作优化原则

  • 只能压缩关键工作。
  • 优先压缩成本低、资源充足的工作。
  • 压缩过程中可能出现多条关键路径，必须同时压缩。
  • 不能将关键工作压成非关键工作。

📝 第三部分：开放式问题答题模板

案例分析题 · 从四个维度系统分析，无标准答案时套用此框架

🔍 1. 问题指出（What's wrong?）

结合案例材料，明确指出项目在哪方面出了问题。

📌 "根据案例描述，该项目在【进度管理 / 成本管控 / 质量管控 / 安全管理 / 风险预判 / 组织管理 / 合同管理】方面存在以下问题：……"

🧠 2. 原因分析（Why does it happen?）

从多个角度分析问题产生的根源：

  • 前期规划不足：可行性研究不充分，目标不清晰。
  • 管理制度不完善：缺乏有效的监督、沟通或考核机制。
  • 人员能力不足：项目经理或团队成员经验欠缺。
  • 风险意识薄弱：未能识别或低估潜在风险。
  • 资源调配不合理：人力、物资、资金未能及时到位。
  • 现场管控不到位：对施工过程的监控不力。

🔧 3. 解决措施（How to fix it?）

针对问题和原因，提出具体、可落地的改进方案：

  • 进度方面：优化施工进度计划，采用网络图科学排期；必要时增加资源或调整工作逻辑关系。
  • 成本方面：建立全过程成本管控体系，使用挣得值分析法动态监控。
  • 质量方面：严格落实三阶段（事前、事中、事后）质量控制，加强工序验收。
  • 安全方面：健全安全管理制度，落实责任到人，加强现场巡检。
  • 风险方面：建立动态风险监控机制，提前制定应急预案。
  • 组织方面：优化组织结构，明确职责分工，提升沟通效率。
  • 合同方面：规范合同与招投标管理，明确双方权责。

✅ 4. 总结（Conclusion）

📌 "综上所述，一个成功的项目必须通过全生命周期、全要素的系统化管理，协调好质量、进度、成本三大核心目标，才能最终实现项目价值。"
